Takes the already allocated GridSquare array grid and sets it up for use by AStarFindRoute and HashMapAStarFindRoute.

Uses objectClearance and movingObjectSize to determine the apparent size of the opponents, and marks the squares that contain them as SquareType::Obstacle.

Marks the square that contains endPoint as SquareType::Destination.

Marks the square that contains startPoint as SquareType::Origin.

Works in parallel as far as possible, using the Microsoft Parallel Patterns Library (http://msdn.microsoft.com/en-us/library/dd492418.aspx)
