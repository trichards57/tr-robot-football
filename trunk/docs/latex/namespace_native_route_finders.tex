Takes the already allocated \hyperlink{class_native_route_finders_1_1_grid_square}{Grid\-Square} array grid and sets it up for use by A\-Star\-Find\-Route and Hash\-Map\-A\-Star\-Find\-Route.

Uses object\-Clearance and moving\-Object\-Size to determine the apparent size of the opponents, and marks the squares that contain them as Square\-Type\-::\-Obstacle.
